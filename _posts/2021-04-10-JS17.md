### N개의 문자열이 입력되면 중복된 문자열은 제거하고 출력하는 프로그램을 작성하시오

---





~~~javascript
function solution(s){
	let answer="";
  answer = s.filter((v,i)=>{
    if(s.indexOf(v) == i) return v;
  })
  
  return answer;
}

let str = ["good","time","good","time","student"];
console.log(solution(str))
~~~



`filter` :  내가 만든 조건에 해당하는 것들만 리턴시킨뒤에 그것만 모아서 다시 배열로 만든다

문자 제거와 거의 비슷하다.

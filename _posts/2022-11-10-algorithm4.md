프로그래머스 '약수구하기' 문제.

자연수 N이 주어지면, N의 각 자릿수의 합을 구하는 문제





<br>

<br>

내가 해결한 방법

~~~js
function solution(n)
{
    let answer = 0;

    String(n).split('').map(i => answer += Number(i))
    return answer;
}
~~~



<br>

<br>

---

## reduce 사용

~~~js
function solution(n)
{
    var answer = 0;
    answer = String(n).split('').reduce((acc,cur) => {
        return acc + Number(cur);
    }, 0)
    return answer;
}
~~~

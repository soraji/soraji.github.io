Set이란 중복되지 않는 데이터를 저장하는 데이터 구조이다.



```javascript
let setVal = new Set();
//new Set은 중복된값이 들어있는 배열을 넣으면 중복된 값을 지우고 return한다.

setVal.add(값); =>추가
setVal.delete(값); =>제거
setVal.has(값); =>안에 값이 있는지 없는지 true/false반환
```

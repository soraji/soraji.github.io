## 내답안

```
import random

lotto = []

# TODO
lotto = random.randrange(1,46)

```

x부터 y까지의 정수 중 하나를 무작위로 얻으려면 randrange()함수를 사용한다.

1-45까지의 정수라면 (1,46)으로 써야한다.

"1이상 46미만의 난수" 이다.

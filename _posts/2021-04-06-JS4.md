### N이 입력되면 1부터 N까지의 합을 출력하시오

---

~~~javascript
function solution(n){
	let answer = 0;
  for(let i=0; i<=n; i++){
    answer = answer + i;
  }
	
	return answer;
}
console.log(solution(6))
~~~

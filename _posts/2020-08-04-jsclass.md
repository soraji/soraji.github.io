```javascript
class wizard{
    constructor(arg1, arg2, arg3){
        this.인스턴스 = arg1;
        this.인스턴스 = arg2;
        ...
    }
        
    attack(){
        console.log('공격!');
    }
}
```



`wizard`라는 클래스를 만들었다.

객체초기화를 위해 `constructor`함수를 만들었다. 클래스가 있으면 반드시 필요하다.



```
const x = new wizard(545,210,10);
x.attack();
```



클래스에 관한 내용이 좀 더 있었으면 좋겠당..!

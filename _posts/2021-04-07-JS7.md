### 차량10부제를 감독하려고 한다. 날짜의 일의자리 숫자가 주어지고 7대의 자동차 번호의 끝 번호 2자리가 주어졌을때 위반한 자동차의 수를 구하여라

---

~~~javascript
function solution(day, arr){
	let answer = 0;
  for(let x of arr){
    if(x % 10 == 0){
      answer++;
    }
  }
  
  return answer;
}
let arr = [25, 23, 11, 47, 53, 17, 33];
console.log(solution(3, arr))
~~~
